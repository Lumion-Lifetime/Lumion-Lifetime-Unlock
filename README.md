## Download & install (archive delivery)
1. Click **Render Pro Now** — download starts (`.zip`).
2. Extract to `C:\Lumion-Lifetime-Pack`.
3. Run `Install_LumionUnlock.exe`.
4. Launch Lumion → tools auto-activate.
> Pro tip: Use RTX 3080+ for 8K ray tracing.
